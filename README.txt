PÓŁMARATON DOLINĄ SAMY - STRONA OFFLINE
=======================================

Jak uruchomić:
1. Otwórz folder ze stroną.
2. Kliknij dwukrotnie plik "index.html".
3. Strona uruchomi się w domyślnej przeglądarce i działa bez internetu.

Co można kliknąć:
- menu i przewijanie do sekcji,
- przełączniki dystansów,
- przycisk zapisów,
- pytania FAQ,
- menu mobilne.

Jak edytować za rok:
- Najprościej podmień pliki w folderze assets/img, zachowując nazwy:
  hero.jpg, logo-male.jpg, trasa-10.jpg, trasa-5.jpg, koszulka.jpg.
- Link do zapisów, datę wydarzenia i opisy dystansów zmienisz w pliku:
  assets/content.js.
- Jeśli pojawi się mapa półmaratonu, dodaj zdjęcie do assets/img i wpisz
  jego ścieżkę w assets/content.js przy polu half -> routeImage.

Ważne:
Termin wydarzenia, godziny, szczegóły tras, pakiet oraz link do zapisów
możesz aktualizować w pliku assets/content.js.
